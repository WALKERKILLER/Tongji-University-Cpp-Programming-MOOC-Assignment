# 同济大学C++程序设计MOOC实验题代码库

## ? 项目概述
本代码库为同济大学《C++程序设计》MOOC课程的实验题解答代码合集，包含课程各章节的编程练习实现。代码经过严格测试，可作为学习参考。

## ? 目录结构
```
HomeWork_mooc1-2/        # 第1-2章实验
  ├─ homework1/          # 实验一
  ├─ homework2/          # 实验二
  └─ homework3/          # 实验三
HomeWork_mooc2-2/        # 第2-2章实验
  ├─ homework1/          
  ├─ homework2/
  └─ homework3/
...（后续章节结构类似）
```

## ? 使用方法
1. 使用Visual Studio打开对应章节的`.sln`解决方案文件
2. 在解决方案资源管理器中右键选择要运行的实验项目
3. 点击"生成" → "生成解决方案"（快捷键Ctrl+Shift+B）
4. 按F5键运行调试程序

## ?? 注意事项
1. 每个实验项目独立运行，切换实验时请重新生成解决方案
2. 代码文件中的`// BEGIN YOUR CODE`到`// END YOUR CODE`之间是核心算法部分
3. 程序运行参数通过"项目属性" → "调试" → "命令参数"设置
