# 同济大学C++程序设计MOOC实验题代码库

## 项目概述
本代码库为同济大学《C++程序设计》MOOC课程的实验题解答代码合集，包含课程各章节MOOC的编程练习实现，可作为学习参考。

## 目录结构
```
HomeWork_mooc1-2/        # 第1-2章实验
  ├─ homework1/          # 实验一
  ├─ homework2/          # 实验二
  └─ homework3/          # 实验三
HomeWork_mooc2-2/        # 第2-2章实验
  ├─ homework1/          
  ├─ homework2/
  └─ homework3/
...（后续章节结构类似）
```
## 注意
请用vs2012及以上版本的vs运行
